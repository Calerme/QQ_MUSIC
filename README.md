# 项目本地运行指南

## 下载项目到本地

```js
git clone git@github.com:Calerme/QQ_MUSIC.git
```

## 安装依赖

```js
npm install
```

## 运行项目与服务

```js
npm run dev
npm run server
```

## 在浏览器输入

localhost:8080

即可浏览。因为 APP 是专为移动端开发的，其中有很多触摸事件，所以请尽量使用开发者工具将浏览模式调为移动端。
